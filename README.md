Сайт управления задачами

Перед запуском сервера устоновите нужные плагины командой:

```bash
npm i
```

Запуск сервера:

```bash
npm run dev
```

При ошибках запуска удалить файл Package-lock.json и повторить команду 

```bash
npm i
```
